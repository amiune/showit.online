Frequently Asked Questions



What is ShowIt.Online?

An automatic way to create a website for a github repo markdowns and jupyter notebooks. See an example here Ideal to share study notes or a mini blog or site.

How do I use ShowIt.Online?

Just write a github username and repo here and get a url to share in a pretty and easy way.

Is ShowIt.Online free to use?

Yes! ShowIt.Online is free and open-source under the MIT License. You can read the code that runs this site here

Why and how can be free forever with no ads?

This project is so simple that needs no servers so it has no cost to mantain.
